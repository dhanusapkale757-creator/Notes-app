# 📝 Notes App

A simple web-based Notes Application that allows users to create, manage, pin, and delete notes easily.  
This project demonstrates basic CRUD operations using HTML, CSS, and JavaScript with Local Storage.

---

## 📌 Project Overview

The Notes App helps users quickly store and organize important information.  
Users can create notes with a title and content, pin important notes, and delete notes when they are no longer needed.

All notes are stored in the browser using Local Storage, so they remain saved even after refreshing the page.

---

## ⚡ How the Application Works

1. User enters title and note content
2. Clicks **Add Note**
3. Note is stored in **Local Storage**
4. Notes appear on the screen as cards
5. User can pin or delete notes anytime

---

## 🖥 User Interface

- Notes appear as cards
- Each card contains:
  - Title
  - Content
  - Pin button
  - Delete button
- Pinned notes are highlighted with a gold border
